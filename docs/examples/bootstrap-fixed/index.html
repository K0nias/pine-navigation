Pine.JS example: Bootstrap fixed inverted

In this test-case:

  • Top fixed navigation build inside Bootstrap layout
  • Bootstrap inverted skin
  • right-to-left effect on small screens
  • Custom small/large breakpoint set to 1200px
  • Implemented via LESS

Pine initialized this way:

$(function() {
    $('.pine').pine({
      fxSmallDisplay: 'fx-collapse',
      largeDisplayStart: '1200px'
    });
  });

Follow comments in css/bootstrap-fixed.less and this HTML file.
