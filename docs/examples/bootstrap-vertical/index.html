Pine.JS example: Bootstrap vertical

In this test-case:

  • Vertical navigation build inside Bootstrap layout
  • Bootstrap skin
  • Custom small/large screen breakpoint
  • More complex navigation structure
  • Implemented via LESS

Pine initialized this way:

$(function() {
    $('.pine').pine({
      fxSmallDisplay: 'fx-collapse',
      largeDisplayStart: '992px'
    });
  });

Follow comments in css/bootstrap-vertical.less and this HTML file.
